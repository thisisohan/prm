# Nemrac Production Management System

**Nemrac Production Management System** is a custom-built web application for **NEMRAC DESIGN LTD.** that streamlines production tracking. Developed using **WordPress, PHP, MySQL, and JavaScript**, the system enables efficient monitoring of production activities across multiple sections and lines, while also ranking the best Section In-Charge and Line In-Charge.

A real-world production management system built for a garment manufacturing company.

---

## Features

- Production Tracking: Monitor production activities across multiple sections and lines.  
- Section & Line Ranking: Automatically rank the best-performing Section In-Charge and Line In-Charge.  
- Role-Based Access: Secure login and access control for Admins, Controllers, Authors, and Employees.  
- Production Logs: Maintain detailed daily logs with flexible filters.  
- Dashboard & Reports: Visualize production data with interactive charts.

---

## Tech Stack

- Backend: PHP (WordPress plugin architecture)  
- Database: MySQL  
- Frontend: HTML, CSS, JavaScript (Chart.js for visualizations)  
- Platform: WordPress
